# Independent
- Basic concept: For any Borel Set $A, B \in \mathbb{R}$ :
$$
P(X \in A, Y \in B) = P(X\in A)P(Y\in B )
$$
- Continuous Type:
$$
\begin{align}
F_{X,Y}(x,y) &= F_X(x)F_Y(y) \\[2ex]
f_P{X,Y} &= f_X(x)f_Y(y) \ a.e.
\end{align}
$$
$$
\begin{align}
Independent &\Rightarrow Cov(X, Y) = 0 \\[2ex] 
Cov(X, Y)= 0 &\nRightarrow Independent \ (not\ gaussian)
\end{align}
$$
