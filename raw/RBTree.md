# RBTree
---
A kind of balance binomial tree that support quick search, insert, 
change, and delete. 

--- 
## Basic Constraints
1. There are only two colors, BLACK and RED
2. The root have to be BLACK.
3. Leaf (nullptr) is BLACK.
4. Path from root to any leaf is equal.
5. Never two RED node become neighbor.

> [!question] Why we define like this?
